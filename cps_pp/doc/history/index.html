An unauthorized history of the Columbia Physics System

This is a short history of CPS. As this is written by someone in UKQCD, this should be viewed as an unauthorized version of events. We believe that understanding the history of a code will help a person learn about the code. Also this document records the people contributing to the project.

The Columbia Physics System was written by the Columbia lattice QCD group to do lattice QCD calculations on the QCDSP computer.

Preparation for QCDOC

Andrew Jackson and Stephen Booth from EPCC converted the code to ANSI C++. (CPS was developed before the ANSI c++ standard). Andy and Stephen also ported the code to run under MPI (Message Passing Interface).

Chris Millar wrote a parallel random number generator. that was node independent. Chris also developed a testing framework for CPS. Zbigniew Sroczynski ported Chris Millar's code back into the main development trunk.

There was a project to update the directory structure and compilation system of CPS. The result of this endeavor is called "cps++". The original proposal for a new code structure for the CPS code, here [html] [pdf] [ps] [source] was written by Andrew Jackson and Balint Joo. The history of the development effort is available.
